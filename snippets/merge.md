# Merge Node

Объединение параллельных потоков данных.

### Пример 1: Combine All (стандартный)
```json
{
  "parameters": {
    "mode": "combine",
    "combineBy": "combineAll"
  },
  "type": "n8n-nodes-base.merge",
  "typeVersion": 3.1
}
```
